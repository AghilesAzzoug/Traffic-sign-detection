# What does it do?
* Detect speed signs
* Detect stop signs

# How?
* Using cascade classifiers for signs detection
* Using multithreading for better performances
* Using flann classifier to detect the speed

# Requirements
* Python >= 3.6.0
* OpenCV >=3.0.0
* A video source

# How to use?
Just start main.py script
